# Customer Service Chatbot

A simple rule-based / NLP-based chatbot that answers basic customer service queries.
This project demonstrates fundamental Natural Language Processing (NLP) concepts and basic chatbot design.

## Features

* Responds to common customer questions
* Rule-based intent matching
* Simple NLP preprocessing
* Lightweight and easy to run
* Command-line chatbot interaction

## Project Structure

```
Customer-Service-Chatbot/
│
├── data/               # Training data / intents
├── src/                # Chatbot logic and NLP processing
├── app.py              # Main application entry point
├── test_bot.py         # Test script for chatbot
├── requirements.txt    # Python dependencies
└── README.md
```

## Installation

1. Clone the repository

```
git clone https://github.com/NANDAKISHORE58/Customer-Service-Chatbot.git
```

2. Navigate to the project folder

```
cd Customer-Service-Chatbot
```

3. Install dependencies

```
pip install -r requirements.txt
```

## Usage

Run the chatbot using:

```
streamlit run app.py
```

Type your question in the terminal and the chatbot will respond with the closest matching answer.

## Example Queries

* "What are your working hours?"
* "How can I contact support?"
* "Where is my order?"
* "What is your return policy?"
